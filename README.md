# multlipicationGame
 YET AGAIN Another fun and interactive random math challenge game created using Small Basic. This time you have to **multiply**
 ---

## 🚀 How It Works

* **Random Generation:** The program automatically generates two random numbers between 1 and 99.
* **Interactive Quiz:** It displays the numbers and asks the player to multiply them.
* **Answer Validation:**
  * If the player's answer is **correct**, it displays a success message.
  * If it is **incorrect**, it reveals the right answer.
* **Game Loop:** At the end of each round, the game asks the player if they want to continue (`Y / N`). The loop keeps running as long as the player enters `Y` or `y`.
